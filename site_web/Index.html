OvoOx Greenhouse Project

Un projet de serre connectée, de sa conception à sa réalisation. qui utilise des capteurs IoT pour collecter des données (température, humidité, CO2) et un modèle de Machine Learning pour prédire et ajuster les conditions optimales de croissance des champignons.

Incluent un volet de recherche sur la culture de champignons.

Le projet

Ce projet vise à développer une serre intelligente qui utilise des capteurs IoT pour collecter des données (température, humidité, CO2) et un modèle de Machine Learning pour prédire et ajuster les conditions optimales de croissance des champignons comestible.

Technologies utilisées : Arduino, Raspberry Pi, Python (TensorFlow), application du Bureau en Java, capteurs environnementaux, plusieurs composantes electroniques et +.
